## ğŸ Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** (v18.0.0 or higher)
- **npm** (v9.0.0 or higher) or **yarn** (v1.22.0 or higher)

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/yourusername/Amplify-AI.git
cd Amplify-AI
```

2. **Install dependencies**

```bash
npm install
```

### Running Locally

1. **Start the development server**

```bash
npm run dev
```

2. **Open your browser**

Navigate to [http://localhost:3000](http://localhost:3000)

The page will automatically reload when you make changes to the source code.

### Available Scripts

- `npm run dev` - Start development server on port 3000
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally

## ğŸ¨ Design System

### Colors

```css
--primary: #8B5CF6;      /* Electric Purple */
--secondary: #FF6B5B;    /* Amplify Coral */
--accent: #38BDF8;       /* Sky Blue */
--lime: #84CC16;         /* Lime Green */
--dark: #020617;         /* Deep Charcoal */
```

### Typography

- **Headings**: Manrope (600, 700, 800)
- **Body**: Inter (400, 500, 600, 700)

### Spacing & Layout

- Max content width: `1280px`
- Section padding: `py-24` (mobile) to `py-32` (desktop)
- Grid system: TailwindCSS responsive grid

### Animation Principles

- **Duration**: 0.3s - 1.5s depending on element size
- **Easing**: Custom cubic-bezier curves for premium feel
- **Stagger**: 0.05s - 0.12s for sequential animations
- **Hover states**: Subtle scale and shadow transformations
